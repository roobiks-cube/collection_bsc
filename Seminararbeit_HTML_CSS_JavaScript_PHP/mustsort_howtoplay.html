Must Sort

How to play




There are different shapes to the left and to the right.

Objective: Sort the image in the middle either to the left or to the right.

Example:



Click "Go" to get the first image. Here the sun should be sorted to the left. Choose "Left" to sort the image to the left.

In the end the buttons will be grayed out except for the "Submit" button. Click "Submit" to save your status.


You can save five games in total for the Must Sort Game 1 and three games for the Must Sort Game 2.
